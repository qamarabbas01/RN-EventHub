# EventHub User Guide

## Getting started

1. **Create an account** from the Register screen.
2. **Log in** with your email and password.
3. Browse **Home** to discover events and **Events** to manage the full list.

## Finding events quickly

- **Search**: Use the search bar to filter events by title.
- **Filters** (Events tab): Tap a filter like **Upcoming / Live / Ended** to narrow results.

## Viewing an event

- Tap an event card to open **Event Details**.
- From Event Details you can:
  - **Edit** the event
  - **Delete** the event (you’ll be asked to confirm)

## Common issues

- **Network error**: Check your connection and try again.
- **Sign-in failed**: Double-check your email/password and try again.
